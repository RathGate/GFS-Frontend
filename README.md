# GFS Vitrine

## Running
Start running this command in a terminal
```shell
cp .env.exemple .env
npm run dev
```

## Installation
- Install [Node.js **v20.9.0**]()
- Install [Composer]()
- Run these commands:
```shell
npm i
```

## Technologies
- [Nuxt.js](https://nuxt.com/)
- [Tailwindcss](https://tailwindcss.com/)
- [Typescript](https://www.typescriptlang.org/)
